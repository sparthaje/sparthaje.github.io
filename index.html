UVA Solar Car Team
On the UVA Solar Car team, we are designing a fully electric car which recharges its batteries via solar panels for the Formula One Grand Prix competition. I'm leading the software operations team which is comprised of two subteams: embedded software and telemetry visualization. The embedded software team has over ten members to develop our C++ distributed embedded system. We are designing software to handle throttle control, develop natural regenerative breaking, data collection, and more! In the near future we will be looking into developing reliable cruise control software. The telemetry visualization team is building a full visualization suite in React for the race pit, using a Python server to handle processing radio messages from the car.
Ascent
Ascent is a research project where I worked in a team 3 to develop a vertical take off and landing aircraft. We wrote the flight software in C++ using the platform.io framework for a Teensy 3.6 flight computer. The attitude was controlled via a thrust vector control system. All the air being pushed out the propellers got funneled into a nozzle which could be angled to angle thrust. We did all the simulations in Simulink in MATLAB for the PID gains. Our position estimating Kalman filter was simulated in Python, and verified in hardware. We fell short of a successful flight as our implementation of the PID loop in simulations failed to match our implementation of the PID loop in the flight software. With the summer after the school year ending and us going off to college, we ended the project with a successful thrust vector system, strong modeling of the physics behind the system, and a reliable state estimating system.
